# 데이터베이스  

- ### SQL workbench

방법 1. SQL문에서 코드로 데이터베이스 만들기
(SQL 코드)  
ex) CREATE SCHEMA `madang` ;

<br>

방법 2. 데이터베이스 이름 작성 -> apply -> 코드 작성
(sql문 코드작성)  
```sql
CREATE SCHEMA `madang` ;
	'bookid' INT NOT NULL,
	'bookname' varchar(40) DEFAULT NULL,
	'publisher' varchar(40) DEFAULT NULL,
	'price' INT DEFAULT NULL,
	PRIMARY KEY ('bookid')
);
```
